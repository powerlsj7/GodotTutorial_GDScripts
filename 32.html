딕셔너리를 배워봄.

사전처럼 특정 문자열을 등록하고 등록된 문자열로 찾아서 저장된 값을 불러올 수 있다.

초기화, 값을 수정하거나 추가하는 방법, for문으로 전체를 탐색하는 방법을 알 수 있다.

extends Node

func _ready():
	#이번에는 Dictionary를 배워봄.
	var players={"Crook":1,"Villain":35,"Boss":100,}
	#출력
	print(players["Villain"])
	#값수정
	players["Villain"]=50
	print(players["Villain"])
	#새로운 키,값 생성
	players["Dwayne"]=999
	print(players["Dwayne"])
	#for문으로 전체 탐색가능
	for username in players:
		print(username+": "+str(players[username]))

이번에는 딕셔너리의 키값 자료형을 달리해봄.

딕셔너리의 자료형도 딕셔너리로 쓸 수 있음.

배열도 가능함.

extends Node

func _ready():
	#자료형을 다르게도 할 수 있다.
	var players={1:1,"Villain":35,0.2:100,}
	for  i in players:
		print(str(i)+" :"+str(players[i]))
	
	var players2={
			"Crook":{"Level":1,"Health" : 80},
			"Villain":{"Level":50,"Health" : 150},
			"Boss" :{"Level":100,"Health" : 500},
			}
	#Boss의 Health의 값을 알고 싶다.
	print(players2["Boss"]["Health"])
	#전체 읽어보기
	for  i in players2:
		print(str(i)+" :"+str(players2[i]))
	#배열을 넣어서 사용할 수 있음.
	var test={"배열1":[1,2,3,4],"배열2":[2,3,4,5]}
	for  i in test:
		print(i+" :"+str(test[i]))
	#결론은 데이터를 구조화하는데 최적화된 클래스
