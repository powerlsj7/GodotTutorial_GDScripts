그림처럼 코드를 작성후에 스페이스바를 5번 누르면 저런 메시지가 뜸. 체력이 0이되면 죽음.



코드를 좀더 수정해서 elif와 else 로 분기를 나눠서 작성하면 그림처럼 50이하일 땐 부상이고 0이면 죽게됨. 

extends Node

var health =100
# Called when the node enters the scene tree for the first time.
func _input(event):
	if event.is_action_pressed("my_action"):
		health-=20
		print(health)
		if(health)<=0:
			health=0
			print("YOU DIED!")
		elif health<50:
			print("YOU ARE INJURED!")
		else:
			print("YOU ARE HEALTHY")
