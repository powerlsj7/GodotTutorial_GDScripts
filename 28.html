이번에는 함수에 매개변수 값을 전달하고 호출해보겠음

값이 잘전달되는 것을 확인할 수 있음.

extends Node
  
  func _ready():
    add(3,8)
    add(245,111)
  func add(num1,num2):
    var result=num1+num2
    print(result)

이번에는 add함수의 반환형식을 void에서 int로 바꾸고 호출하는 코드를 작성해봄.

result변수값이 달라지는 것과 add함수가 int형 변수를 리턴하는 것을 확인할 수 있음.

extends Node

func _ready():
	var result=add(3,5)
	print(result)
	result=add(result,10)
	print(result)
func add(num1 : int,num2 : int)->int:
	var result=num1+num2
	return result
